|  5  |  11  |  GPIO 5  |
|  -  |  5  |  SD_CLK  |
|  -  |  9  |  SD_D0  |
|  -  |  7  |  SD_D1  |
|  -  |  6  |  SD_D2  |
|  -  |  3  |  SD_D3  |
|  -  |  8  |  SD_CMD  |
|  12  |  16  |  GPIO 12  |
|  13  |  18  |  GPIO 13  |
|  14  |  17  |  GPIO 14  |
|  15  |  19  |  GPIO 15  |
|  16  |  15  |  GPIO 16  |

The micropython pin value is the one used as parameter for machine.Pin class initialisation
